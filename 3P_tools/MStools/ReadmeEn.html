Information regardintg the content of the folder SpbVipTools\3P_Tools\MSTools

It is expected, that this folder contains Microsoft files to build libraries:

lib.exe

mspdb71.dll

which are the part of the Visual Studio.

If the system Visual Studio is installed at your computer, then you may copy these files to this folder or you must set appropriate path to the file lib.exe in those cmd files, which are responsible for the building libraries (.lib files).
